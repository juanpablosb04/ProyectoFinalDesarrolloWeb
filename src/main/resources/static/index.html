Nuestros Productos

Variedad de flores y plantas para el hogar, contamos con arreglos para diferentes ocasiones especiales, ademas de productos para sus plantas y el cuidado de ellas.

Origen de nuestros productos

Todos nuestros productos son plantados y cuidados por nosotros para asegurar la calidad y larga durabilidad de cada uno.

Origen

Polem y Aromas es una floristería fundada por dos hermanos que tuvieron la idea de crear una pequeña floristería para su tiempo libre y que poco a poco a ido creciendo con la ayuda de nuestros clientes.
